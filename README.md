# Життя — Симулятор фінансового балансу

Інтерактивний веб-симулятор для розрахунку щомісячного бюджету українця залежно від міста, типу зайнятості, житла, транспорту та стилю життя.

## Демо

[![Демо на YouTube](https://img.youtube.com/vi/t-mVwheSYxM/maxresdefault.jpg)](https://youtu.be/t-mVwheSYxM?si=SYfQBcFlWRDmJyv7)

## Що це

Користувач обирає параметри свого життя — місто, зарплату, форму зайнятості, оренду/своє житло, транспорт, тварин, підписки, харчування в кафе тощо — і бачить реальний залишок грошей наприкінці місяця з детальним розбивом витрат.

## Технології

- **Next.js 14** — фреймворк
- **React 18** + TypeScript
- **Tailwind CSS** — стилізація
- **Jest** — тести
- **i18n** — локалізація (UA/EN)

## Запуск локально

```bash
npm install
npm run dev
```

Відкрий [http://localhost:3000](http://localhost:3000).

## Скрипти

| Команда | Дія |
|---|---|
| `npm run dev` | Запуск dev-сервера |
| `npm run build` | Продакшн білд |
| `npm start` | Запуск продакшн-сервера |
| `npm test` | Запуск тестів |
| `npm run test:watch` | Тести у watch-режимі |

## Параметри симулятора

- **Місто** — Київ, Львів, Дніпро, Тернопіль, Село
- **Зайнятість** — найманий (офіц./конверт), ФОП 1/2/3 гр., Дія.City
- **Житло** — оренда, своє, у батьків
- **Транспорт** — громадський, авто, таксі, мікс, пішки
- **Тварини**, **підписки**, **харчування у кафе**, **соціальне життя**
